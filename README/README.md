# Student Portfolio & Academic Management Website


## Project Overview

This project is a responsive multi-page portfolio website developed as part of my **Data Science coursework at Miva Open University**. It serves as both a personal academic portfolio and a student academic management platform, showcasing my profile, technical skills, projects, and an interactive academic planner.

The website was designed using **HTML5, CSS3, and JavaScript**, following modern web development practices with a responsive and user-friendly interface.

## How to Run the Project

1. Clone this repository:

```bash
git clone https://github.com/captainstark15/student-portfolio.git
```

2. Open the project folder in Visual Studio Code.

3. Install the **Live Server** extension (recommended).

4. Right-click `index.html` and select **Open with Live Server**.

Alternatively, open `index.html` directly in your web browser.
